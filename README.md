# 🌐 Vinayak Kanavalli — Portfolio

![Portfolio Preview](assets/img/about.jpg)

A clean, responsive personal portfolio website built with **HTML, CSS, and JavaScript** — showcasing projects, skills, certificates, and contact information.

🔗 **Live Site**: [vinayak-ck.github.io/MyPortfolio](https://vinayak-ck.github.io/MyPortfolio/)

---

## 📌 Sections

- **Hero** — Introduction and quick links
- **About** — Background, CGPA, and experience
- **Skills** — Programming languages, IT concepts, technologies
- **Qualification** — Education timeline
- **Certificates** — 11 verified certificates
- **Projects** — 5 featured projects with live demos
- **Contact** — Functional contact form (email delivery via FormSubmit)

---

## 🛠️ Tech Stack

| Layer | Tech |
|---|---|
| Markup | HTML5 |
| Styling | CSS3 (custom, responsive) |
| Interactivity | Vanilla JavaScript |
| Charts/Animations | ScrollReveal, Swiper.js |
| Contact Form | FormSubmit |
| Hosting | GitHub Pages |

---

## 🚀 Run Locally

```bash
git clone https://github.com/vinayak-ck/MyPortfolio.git
cd MyPortfolio
# Open index.html in your browser
open index.html
```

No build step required — it's pure HTML/CSS/JS.
